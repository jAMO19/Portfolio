Project Overview

With climate change increasing tree mortality rates across British Columbia, this project develops an automated workflow using QGIS Model Builder to assess tree mortality risk across BC. The model combines multiple spatial datasets to identify high-risk areas, enabling community protection planning.

By automating the analysis through Model Builder, this workflow can be easily updated with new data or applied to different regions, making it a scalable solution for ongoing tree mortality analysis.

QGIS Model Builder Workflow

The automated model processes multiple spatial datasets through a series of geoprocessing steps, applying summarize overlay analysis to generate a comprehensive risk assessment. This approach ensures consistency, repeatability, and enables rapid updates when new data becomes available.
